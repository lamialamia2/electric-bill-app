# ⚡ বিদ্যুৎ বিল হিসাব — React App

## কীভাবে চালাবেন

### ১. Node.js ইনস্টল করুন
https://nodejs.org থেকে Node.js (v18+) ডাউনলোড করুন।

### ২. ডিপেনডেন্সি ইনস্টল করুন
```bash
cd electric-bill-app
npm install
```

### ৩. অ্যাপ চালু করুন
```bash
npm start
```
ব্রাউজারে http://localhost:3000 খুলবে।

### ৪. Build (production)
```bash
npm run build
```

## ফিচার
- মিটার রিডিং থেকে অটো ইউনিট হিসাব
- মোট টাকা ÷ মোট ইউনিট = প্রতি ইউনিট দাম
- প্রত্যেকের ইউনিট × প্রতি ইউনিট = দেওয়ার কথা
- আগের মাসের ব্যালেন্স যোগ করার সুবিধা
- কে কত কম/বেশি দিয়েছে তা দেখানো
